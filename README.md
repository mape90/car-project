car-project
===========

States
======
1 Wait_state
2 running_stete
3 find_road_state
4 road_not_found_state
 


Motor 
=======
 * PH3 (Timer4)
 * INA - PK0 (ADC8)
 * INB - PK1 (ADC9)
 * ENA - PK2 (INPUT)
 * ENB - PK3 (INPUT)

Servo 
=======
 * Timer1
 * PWM - PB5 (OC1A)

Tachometer 
=======
 * Timer5
 * Signal - PL2 (T5)


LCD
=======
 * RX - PD2 (RXD1)
 * TX - PD3 (TXD1)

Led
=======
 * PC0 - led0 
 * PC1 - led1
 
Button 
=======
 *(HIGH when not pushed)
 * PE5 (INT5)
